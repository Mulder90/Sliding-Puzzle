### Sliding Puzzle

Final project for **AI** class (Prof. Frasconi Paolo) at the University of Florence.  
Solves 8 and 15 puzzle with:
- **A***  
- **IDA***  
- **Weighted A***   
- **BFS** (Only 8 puzzle) 
- **Iterative DLS** (Only 8 puzzle)  

and heuristics:
- **Manhattan**  
- **Misplaced**  

**N.B:** Goal state at bottom right position.
